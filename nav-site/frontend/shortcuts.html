键盘快捷键帮助

管理

首页快捷键

  • Alt + S
    聚焦搜索框
  • Alt + A
    显示所有网站
  • Alt + M
    进入管理页面
  • Esc
    清空搜索框
  • 1-9
    快速选择前9个父分类

管理页面快捷键

  • Alt + W
    切换到网站管理
  • Alt + C
    切换到分类管理
  • Alt + S
    切换到设置
  • Alt + A
    添加新网站/分类（根据当前标签）
  • Alt + E
    导出数据
  • Alt + I
    导入数据
  • Esc
    取消当前操作（关闭表单）
